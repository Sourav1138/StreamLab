# StreamLab Cinema Engine v2.0

StreamLab is a Flask-based web application that allows you to stream video files or Google Drive links directly to your browser using FFmpeg for real-time transcoding.

## Installation

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Install FFmpeg.

## Usage

Run `python app.py` and navigate to `http://127.0.0.1:5000`.
